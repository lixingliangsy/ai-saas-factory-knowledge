# 阶段 07 转化闭环草案（触达 → 承接 → 成交 → 回流）

> 设计人：转化路径设计师 闭环（opc-conversion-designer）
> 边界声明：本草案只设计"路径结构"——每一层的机制、artifact 占位、指标与衔接。不写广告文案、不写 LinkedIn 帖子正文、不写落地页完整正文（留给资产沉淀/运营阶段）。正文处统一用「〔…〕」占位。

## 0. 核心约束与设计前提

- 无私域 [假设-高置信]：不能依赖微信群/私聊培育。
- 低预算 [假设-高置信]：近零边际成本，自助化。
- 英文销售弱 [假设-高置信]：不依赖 1:1 人工销售，靠产品价值与内容自转化。
- ⇒ 路径必须是「自助 + 内容/产品驱动 + 近零边际成本」。
- **#1 决定性依赖 [假设-待确认]：最懂人群 = 海外科研人员。若此假设被推翻，整条利基与转化路径需重做。**

## 1. 触达层（Reach）：入口角色与分工

四个入口，按"成本—预期—诚实边界"列示。建议收缩到 1–2 个主入口（见末尾待确认）。

| 入口 | 角色 / 负责带谁进来 | 内容 / 动作类别 | 成本 | 预期 | 诚实边界 |
|---|---|---|---|---|---|
| A. GEO 内容引擎 | 高意图科研人员（主动搜"如何做可发表图表 / 某期刊配图规范"） | 方法指南、期刊配图规范对比页、how-to（Nature/Science/Cell 图例要求） | 低（内容生产，分发近零） | 慢热但复利；主获客驱动 | GEO 新兴渠道，效果不可夸大/不可保证即时流量 |
| B. 产品免费层口碑 | 实验室同伴（免费用户分享出图成果 → 同侪发现） | 产品内分享链接、水印引流、输出物自带品牌 | 近零边际（产品已上线） | CAC 最低；复购/转介闭环 | 依赖产品留存质量；冷启动需其它入口 seeding |
| C. LinkedIn 自动发布 | 科研相关弱连接（博后 / PI / 实验室管理者）信任沉淀 | 创始人洞察、产品更新、幕后；用排程工具降低人工 | 低（排程工具） | 信任 / 品牌，直接转化偏低 | 弱英文销售下深度有限；转化弱于 A |
| D. 学术社区（可选 / 暂缓） | 垂直高意图小众流量（Reddit r/labrats、ResearchGate、学术 X、StackExchange） | 非 spam 的实质参与、答疑、资源帖 | 时间成本；spam 风险 | 小众高意图；测试期再上 | 人手有限，先做 A/B/C，D 留待信号确认后 |

**主入口建议**：GEO（A）+ 产品免费层（B）为主，LinkedIn（C）为信任层，学术社区（D）为放大期可选测试。收缩依据：低预算 + 弱销售 + 无私域，只能押注"复利型、自助型"入口。

## 2. 承接层（Capture / Engage）：访客 → 注册 / 试用 / 留资

因无私域、无人工销售，承接必须自助、轻量、无 1:1 私聊。

### 2.1 机制
- **免费层注册（主承接）**：落地 → 创建免费账号 → 获有限免费额度 / 低清带水印出图。
- **极简落地页**：单页转化，含「〔价值主张一句〕〔价格〕〔主CTA按钮〕〔免费层次CTA〕」。形态③=真页面；形态②=假门（按钮仅收集邮箱）。
- **Waitlist（次承接 / 假门 / 新功能）**：未就绪功能或新期刊模板包，仅收邮箱。
- **产品内引导（aha）**：注册后引导"上传数据 → 选模板 → 出图"首图流程，快速触发价值感，提升 free→paid。
- **轻量自动培育（无销售）**：欢迎邮件序列（3 封，纯价值、无销售话术），靠模板化、自动化，不靠人。

### 2.2 各环节承接目标与 artifact 占位
- 极简落地页（形态③真 / 形态②假门）：artifact = `landing.html`〔价值主张一句 + 价格 + 主CTA + 免费层次CTA〕
- 免费注册 / 账号：artifact = 〔注册表单占位：email / Google OAuth〕
- Waitlist 收集：artifact = 〔邮箱收集组件占位〕
- 产品内首图引导：artifact = 〔引导步骤占位：上传 → 选模板 → 出图〕
- 欢迎邮件序列：artifact = 〔3 封自动邮件占位：欢迎 / 技巧 / 模板掉落，无销售〕

## 3. 成交层（Close）：免费 → 首次付费

结合阶段 04「scifigure-pro 单点付费（积分制）」首验。

### 3.1 定价呈现
- 透明积分包：免费 = N 次低清 / 带水印出图；付费 = 高清 / 去水印 / 期刊模板积分。
- 落地页与产品内同步展示三档占位：〔免费 / 积分包 / 课题组订阅〕。

### 3.2 支付嵌入
- Waffo 优先（真实收款），Stripe 备用；嵌在"付费触发点"处——产品内额度触顶时 + 落地页"购积分"按钮。
- artifact：〔结账组件占位：Waffo/Stripe 嵌入式，仅需必要法律文案〕

### 3.3 free → paid 转化触发设计（触发点）
1. **免费额度用尽**：出图次数耗尽 → 提示充值。
2. **高清 / 去水印解锁**：想导出干净图 → 需积分。
3. **期刊专属模板包**：Nature / Science / Cell 模板 = 付费。
4. **批量 / 加速渲染**：重度用户付费档。
- 最高意图时刻 = "刚出好一张图、想导出干净版" → 此刻弹付费墙 + 一键结账。

### 3.4 成交 artifact 占位
- 付费墙弹窗：artifact = 〔额度用尽提示 + 解锁权益 + Waffo/Stripe 按钮〕
- 定价页：artifact = 〔免费 / 积分 / 订阅三档占位〕

## 4. 闭环回流（Loop）：让路径成环

- **复购 / 订阅**：积分耗尽 → 复充；重度用户 → 课题组订阅。
- **口碑自传播**：免费用户分享出图 → 同侪发现 → 新免费注册（产品驱动闭环，CAC≈$0）。
- **GEO 内容反哺**：真实用户问题、模板需求、用例 → 成为 GEO 文章选题 → 更多触达；获授权用户出图 → 社媒证据内容。
- **公开评价循环**：引导 ProductHunt / 学术社区公开评价 → 信任回流。

## 5. 分阶段指标（与阶段 04 漏斗 + 阶段 06 阈值对齐）

| 层 | 指标 | 健康信号 | 预警信号 | 对齐阶段 06 阈值 |
|---|---|---|---|---|
| 触达 | GEO / LinkedIn / 转介曝光 | 稳定爬升 | 4–6 周仍 0 自然流量 | — |
| 落地访问 | 合格访问数 | 持续流入 | 断流 | B：≥30 合格访问(信号) / ≥50(强PASS)；C：月访问 ≥200 |
| 注册率 | 注册 / 访问 | ≥20–30%（低摩擦） | <10% | — |
| 首付转化 | 真付款 / 注册（或 / 合格访问） | 有真实付款 | 0 付款 | B：≥1 真付款(信号) / ≥3(强PASS)；C：转化 ≥2% |
| 复购留存 | 复充 / 订阅数 | 积分耗尽复充 | 0 复购 | — |
| CAC | 获客成本 | ≈$0（自助 / 转介） | 出现付费买量 | C：CAC≈$0 |

## 6. 与 MVP 形态的衔接

- **形态③（真部署 + 真支付 + 极简落地页 + 极小真实流量）= 首验主路径（B 步）**：触达 = 最小 GEO + LinkedIn + 产品免费层（已上线）；承接 = 真极简落地 + 免费注册；成交 = 真 Waffo/Stripe 结账（真实付费墙触发）。直接承载 B 步阈值。
- **形态②（假门落地页）= 并行 sanity-check**：同渠道但落地页为假门（按钮仅收邮箱，无真支付），验证点击意图，先于 / 伴随③降低风险。
- **放大期（C 步）才补**：学术社区主动运营、完整定价档、订阅计费自动化、邮件培育深化、GEO 内容放量。

## 8. 三条候选转化路径 + 4. 我有自己的方案

**路径 A（推荐主结构 · 内容 / GEO 驱动自助）**
GEO 文章（科研人员搜"如何做 Nature 级配图"）→ 极简落地 / 免费注册 → 产品内首图(aha) → 导出时付费墙(积分) → Waffo 结账 → 复购 + 转介 + GEO 选题反哺
- 适用：低预算、无销售、要复利型 organic。
- 优点：近零边际成本、CAC≈$0、长期复利。
- 代价：起量慢；GEO 新兴不可保证；内容需持续投入。

**路径 B（产品驱动 / 免费层口碑优先）**
现有免费用户分享出图 → 同侪经链接 / 水印发现 → 免费注册 → 付费墙 → 结账 → 闭环
- 适用：若产品留存好，想极致低 CAC。
- 优点：CAC 最低、纯产品闭环。
- 代价：冷启动依赖 A / LinkedIn seeding；完全受产品留存质量约束。

**路径 C（LinkedIn 信任 + 直达免费层）**
LinkedIn 创始人帖 → 科研人员关注 → 访问落地 → 免费注册 → 付费墙
- 适用：建信任 / 权威、科研相关弱连接。
- 优点：信任沉淀、内容资产。
- 代价：直接转化偏低；弱英文销售下深度有限。
